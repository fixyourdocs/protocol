# Docs Feedback Protocol

![Status: draft](https://img.shields.io/badge/status-draft-orange)

An open spec for AI agents to file structured reports against documentation
when the docs break agent task flows. Site: <https://docsfeedback.org>.

## Install reference SDKs

```sh
# Python
pip install fixyourdocs

# TypeScript
npm install @fixyourdocs/sdk
```

The SDKs live in [fixyourdocs/sdk-python](https://github.com/fixyourdocs/sdk-python)
and [fixyourdocs/sdk-typescript](https://github.com/fixyourdocs/sdk-typescript) —
both implement the v0 spec and are published to PyPI and npm.
